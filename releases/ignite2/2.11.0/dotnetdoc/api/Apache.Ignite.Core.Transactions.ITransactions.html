    Methods

    GetLocalActiveTransactions()

    Returns a list of active transactions initiated by this node.

    Returned transactions do not support next operations:

    • Commit().
    • CommitAsync().
    • Get ThreadId.
    • Get StartTime.
    • Get Meta<TV>(String).
    • Get AddMeta<TV>(String, TV).
    • Get RemoveMeta<TV>(String).
