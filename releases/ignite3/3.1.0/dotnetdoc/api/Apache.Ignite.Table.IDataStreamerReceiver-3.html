ReceiveAsync(IList<TItem>, TArg, IDataStreamerReceiverContext, CancellationToken)

Receives a page of items from the data streamer.

The receiver is called for each page (batch) in the data streamer and is responsible for processing the items, updating zero or more tables, and returning a result. 
ValueTask<IList<TResult>?> ReceiveAsync(IList<TItem> page, TArg arg, IDataStreamerReceiverContext context, CancellationToken cancellationToken)
