    public class TcpDiscoveryGoogleStorageIpFinder
extends TcpDiscoveryIpFinderAdapter
    Google Cloud Storage based IP finder.

    For information about Cloud Storage visit cloud.google.com.

    Configuration

    Mandatory

    • Service Account Id (see setServiceAccountId(String))
    • Service Account P12 key file path (see setServiceAccountP12FilePath(String))
    • Google Platform project name (see setProjectName(String))
    • Google Storage bucket name (see setBucketName(String))

    Optional

    • Shared flag (see setShared(boolean))

    The finder will create a bucket with the provided name. The bucket will contain entries named like the following: 192.168.1.136#1001.

    Note that storing data in Google Cloud Storage service will result in charges to your Google Cloud Platform account. Choose another implementation of TcpDiscoveryIpFinder for local or home network tests.

    Note that this finder is shared by default (see TcpDiscoveryIpFinder.isShared().
