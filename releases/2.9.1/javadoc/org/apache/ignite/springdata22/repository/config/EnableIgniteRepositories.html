org.apache.ignite.springdata22.repository.config

Annotation Type EnableIgniteRepositories


  • @Target(value=TYPE)
 @Retention(value=RUNTIME)
 @Documented
 @Inherited
 @Import(value=IgniteRepositoriesRegistar.class)
public @interface EnableIgniteRepositories
    Annotation to activate Apache Ignite repositories. If no base package is configured through either value(), basePackages() or basePackageClasses() it will trigger scanning of the package of annotated class.
