  • public class KafkaStreamer<K,V>
extends StreamAdapter<org.apache.kafka.clients.consumer.ConsumerRecord,K,V>
    Server that subscribes to topic messages from Kafka broker and streams its to key-value pairs into IgniteDataStreamer instance.

    Uses Kafka's High Level Consumer API to read messages from Kafka.
