Class Pipeline<K,V,C extends Serializable,L>

  • java.lang.Object
    • org.apache.ignite.ml.pipeline.Pipeline<K,V,C,L>
  • Type Parameters:
    K - Type of a key in upstream data.
    V - Type of a value in upstream data.
    All Implemented Interfaces:
    Serializable

    public class Pipeline<K,V,C extends Serializable,L>
extends Object
implements Serializable
    A simple pipeline, which acts as a global trainer which produce a Pipeline Model. A Pipeline consists of a sequence of stages, each of which is either a Preprocessing Stage or a Trainer. When fit() method is called, the stages are executed in order.
