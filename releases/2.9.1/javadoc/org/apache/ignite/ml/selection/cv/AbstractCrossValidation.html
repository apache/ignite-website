    public abstract class AbstractCrossValidation<M extends IgniteModel<Vector,Double>,K,V>
extends Object
implements Serializable
    Cross validation score calculator. Cross validation is an approach that allows to avoid overfitting that is made the following way: the training set is split into k smaller sets. The following procedure is followed for each of the k “folds”:
    • A model is trained using k-1 of the folds as training data;
    • the resulting model is validated on the remaining part of the data (i.e., it is used as a test set to compute a performance measure such as accuracy).
