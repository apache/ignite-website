org.apache.ignite.plugin.extensions.communication

Interface MessageFactoryProvider

  • All Superinterfaces:
    Extension, MessageFactory

    public interface MessageFactoryProvider
extends MessageFactory
    Provider of communication message factories.

    Implementation of this interface is responsible for registration of all message factories in registerAll(org.apache.ignite.plugin.extensions.communication.IgniteMessageFactory) method.

    registerAll(org.apache.ignite.plugin.extensions.communication.IgniteMessageFactory) method's call is responsibility of IgniteMessageFactory implementation.
