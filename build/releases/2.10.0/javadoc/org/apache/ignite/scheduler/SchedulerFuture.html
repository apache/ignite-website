org.apache.ignite.scheduler

Interface SchedulerFuture<R>

  • All Superinterfaces:
    IgniteFuture<R>

    public interface SchedulerFuture<R>
extends IgniteFuture<R>
    Future for cron-based scheduled execution. This future is returned when calling IgniteScheduler.scheduleLocal(Callable, String) or IgniteScheduler.scheduleLocal(Runnable, String) methods.
