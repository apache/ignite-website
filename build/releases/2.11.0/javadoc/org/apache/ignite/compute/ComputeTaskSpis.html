org.apache.ignite.compute

Annotation Type ComputeTaskSpis


  • @Documented
 @Retention(value=RUNTIME)
 @Target(value=TYPE)
public @interface ComputeTaskSpis
    This annotation allows task to specify what SPIs it wants to use. Starting with Ignite 2.1 you can start multiple instances of LoadBalancingSpi, FailoverSpi, and CheckpointSpi. If you do that, you need to tell a task which SPI to use (by default it will use the fist SPI in the list).
